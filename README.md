# Blog application created with React

### Steps in order to run:
#### 1. Clone the repository 
Wherever you take care of your Git commands:
```
git clone https://github.com/NatanKwaskiewicz/BlogReact.git
```
#### 2. Open the project in an editor of your choice and install node_modules
In terminal: 
```
npm install
```
or 
```
npm i
```
#### 3. Run the project
In terminal:
```
npm run dev
```
#### 4. See the application in your browser
You can either click on the link generated by Vite in terminal or type the following text into you browser's search bar:
```
localhost:<port>
```
Where <port> is generated by Vite and might vary. Usually it will be 5173.
